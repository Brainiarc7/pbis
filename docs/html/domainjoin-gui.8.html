Name

domainjoin-gui — Graphically join a host to an Active Directory domain

Synopsis

domainjoin-gui

Description

The domainjoin-gui command launches a graphical application that provides a one-step method of joining Linux, Unix, and OS X hosts into an Active Directory domain. The tool performs all the system management configuration necessary to enable local PAM aware services so that they will be capable of authenticating AD domain user login requests.

Debugging Information

In the event that the local host machine fails to successfully join the AD domain, domainjoin-gui dialog will allow you to save the log file containing the verbose debugging information obtained during the join attempt. This information may then be used to obtain support from Likewise Software or the Likewise Open community mailing lists.

See Also
