2.はじめに
  2.1.  本書の記述に関して
      本書関数仕様にて、関数名および関数説明にて記述されている(MB)/(WC)は以下の
      意味をあらわします。
     （ただしWCは一部サポートしておりません）
      MB : Multi Byte 
      WC : Wide Charactor
      また、関数仕様は以下の記述規定によって記述されています。
      例）  2.1.OpenSOAP Class・・・・・・・・・・・・・クラス名
                記述ファイル：OpenSOAP.h・・・・・・・・当関数記述ヘッダファイル名
                使用ヘッダファイル：<Defines.h> ・・・・当ヘッダファイルにて参照し
　　　　　　　　　　　　　　　　　　　　　　　　　　　　ているヘッダファイル名

                2.1.1.OpenSOAP API 初期化 ・・・・・・・関数概要
                      関数名OpenSOAPInitialize・・・・・関数名
                      概  要 OpenSOAP APIを使用する際の各初期化処理を行います。
                      形  式 OpenSOAPInitialize(param)・使用形式
                      仮引数 名前       型         I/O  説明
                             param      void *     I    
                                            ・・・・・・仮引数情報
                                                        I : Input Paramater
                                                        O : OutPut Paramater
                      戻  値 型         説明
                             int        エラーコード
                                            ・・・・・・戻値情報
