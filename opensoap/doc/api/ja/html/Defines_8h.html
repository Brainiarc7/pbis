マクロ定義

#define OPENSOAP_CLASS
 

OpenSOAP API で C++ のクラス定義に付加するマクロ.

このマクロは、クラスのメンバ関数すべてを export したい場合に使用する。 例: 

        class OPENSOAP_CLASS TwoChannel {
        public:
                TwoChannel();
                const std::string &getOne() const;
        };
